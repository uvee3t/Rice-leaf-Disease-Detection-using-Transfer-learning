# Rice-leaf-Disease-Detection-Using-Transfer-Learning
Live app  - https://rice-leaf-disease-detection-using-transfer-learning-b7cywy5dhs.streamlit.app


Rice Leaf Disease Detection using Deep Learning and Transfer Learning, where we built an intelligent system that can automatically analyze rice leaf images and identify diseases with high accuracy.

🚀 Live AI Application: https://rice-leaf-disease-detection-using-transfer-learning-b7cywy5dhs.streamlit.app
 
🌾 Explainable Attention-Guided Transfer Learning Framework for Rice Leaf Disease Detection
Crop diseases remain one of the major challenges affecting agricultural productivity worldwide. Early and accurate detection can significantly improve crop yield and support farmers in making timely decisions.
To address this real-world challenge, I developed an AI-powered rice leaf disease detection system that can identify diseases directly from simple leaf images captured using normal mobile cameras.
Why it is Good  and Different ?
Unlike many deep learning systems that require high-quality or HD images and large datasets, this framework performs effectively with small datasets and standard-quality images, making it practical for real-world agricultural environments and farmers.
Author - Yuvraj Sharma
 
🔬 Research Inspiration
This research work is influenced by the study:
https://doi.org/10.1016/j.inpa.2024.04.006
Building upon this research, I implemented a framework that combines transfer learning, attention-based learning, ensemble modelling, and explainable AI techniques to improve both model accuracy and interpretability.
 
🤖 Deep Learning Models Used
The framework explores multiple state-of-the-art convolutional neural network architectures:
• AlexNet + Custom CNN (Base Model)
• VGG16 (Best Performing Model)
• ResNet50
• MobileNetV2
• EfficientNetB0
• Ensemble CNN Model
These models were trained to detect key rice leaf conditions:
✔ Bacterial Leaf Blight
✔ Brown Spot
✔ Leaf Smut
✔ Healthy / Non Rice Leaf
The deployed system achieves ~96% to 99% prediction accuracy using deep learning .
 
🧠 Techniques Used in the Framework
🔹 Transfer Learning
Leveraging pretrained deep learning models to achieve strong performance even with limited agricultural datasets.
🔹 Attention-Guided Learning
The model learns to focus on the most relevant disease regions of the leaf, improving classification reliability.
🔹 Explainable AI (XAI)
Integrated Grad-CAM heatmap visualization to highlight the regions of the leaf influencing the model’s predictions, making the AI system transparent and interpretable.
 
🌱 Why This Framework is Important
✔ Works with simple mobile images
✔ Effective even with small datasets
✔ Provides visual explanation of predictions using Grad-CAM
✔ Supports early disease detection for farmers
✔ Enables AI-driven precision agriculture
This makes the technology accessible, scalable, and practical for real-world agricultural deployment.
